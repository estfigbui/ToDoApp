# To Do App
*Android application for the creation of a to-do list of tasks.*

## List of implementations
* Jetpack Compose.
* MVVM and Clean architecture.
* Room.
* Dependency injection with Dagger Hilt.
* Flow.
* Live Data.
* RecyclerView.
